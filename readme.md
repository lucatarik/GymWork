🎖️ V-Tracker Pro: Sergente Hartman Edition
"Sveglia larva! Se non tieni traccia dei tuoi progressi, stai solo perdendo tempo a farti i muscoli di gelatina!"

V-Tracker Pro è un'applicazione web leggera, brutale ed efficiente per monitorare i tuoi allenamenti in palestra. Niente fronzoli, niente pubblicità, solo tu, il ferro e il cronometro.

🛠️ Caratteristiche Principali
Gestione Schede Multi-Giorno: Organizza i tuoi allenamenti (Giorno A, B, C...) con facilità.

Contatore Set "Combat": Ogni volta che finisci una serie, clicca e scala il numero. Quando arrivi a zero, l'esercizio è completato.

Timer di Recupero Integrato: Un timer rapido (30s, 60s, 90s, 120s) per non battere la fiacca tra una serie e l'altra.

Reset Intelligente: Quando finisci tutti gli esercizi della giornata, appare il bottone di reset che pulisce la sessione e salva la data dell'ultimo completamento.

Modalità Modifica (Edit Mode): Modifica nomi, esercizi, set, ripetizioni e pesi direttamente dall'interfaccia.

Import/Export JSON: Salva la tua scheda o importane una nuova incollando una stringa di testo.

Hartman Voice: Se il tuo browser lo supporta, verrai motivato (a modo mio) durante l'allenamento.

🚀 Installazione Rapida
Non c'è niente da installare. È un unico file HTML autonomo.

Scarica il file index.html.

Aprilo con qualsiasi browser (Chrome, Safari, Firefox).

Inizia a spingere!

📋 Come Funziona
Imposta la tua scheda: Clicca sull'icona della matita (Edit Mode) in alto a destra per aggiungere giorni ed esercizi.

Allenati: Clicca sul numero di set rimanenti per scalarli. Quando un esercizio arriva a 0, diventa verde.

Recupera: Usa il timer in alto. Clicca sul tempo per farlo partire. Se vuoi fermarlo, clicca di nuovo.

Chiudi la sessione: Una volta terminati tutti gli esercizi, clicca sul bottone "✅ RESET DEL GIORNO". Questo registrerà la data odierna sotto il nome del giorno e resetterà i set per la prossima volta.

💾 Salvataggio Dati
Tutti i dati (la tua scheda, i tuoi progressi e le date) vengono salvati nel LocalStorage del tuo browser. Se cancelli la cache del browser o cambi dispositivo, i dati andranno persi, quindi usa la funzione Export JSON per farti un backup ogni tanto!
